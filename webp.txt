Google은 온라인 이미지의 크기를 줄이기 위해 WebP 파일 포맷을 개발했습니다. 
사진과 그래픽의 크기를 줄여 웹 사이트가 빠르게 로드되고 사용자에게 더 나은 경험을
제공하는 WebP 파일 포맷의 기능을 자세히 알아봅니다.
WebP은 인터넷에서 이미지가 로딩되는 시간을 단축하기 위해 Google이 출시한 파일 포맷입니다. 
WebP를 사용하면 웹 사이트에서 고품질 이미지를 표현할 수 있지만 PNG, JPEG 등 기존 포맷보다 
파일 크기가 작아집니다.


하지만 WebP 파일을 저장할 때 압축 방식(무손실 압축, 손실 압축)을 선택할 수 있으므로 
데이터 손실 없이 또는 중요하지 않은 정보의 일부만 손실하면서 이미지를 압축할 수 있습니다. 
Google 개발자는 무손실 WebP 이미지가 PNG보다 최대 26%까지 줄어들 수 있다고 강조합니다.
웹 사이트 이미지 크기 축소
WebP는 웹 사이트 이미지의 품질과 실용성 사이에서 균형을 유지합니다. 
WebP 파일은 예측 코딩 기술을 기반으로 이미지 품질을 그대로 유지하면서 사진과 그래픽을 압축합니다. 
또한 이미지 크기가 작을수록 사용자의 광대역 네트워크 연결에 대한 부담이 줄어들고 웹 사이트 탐색 속도가 빨라져 
웹 경험이 향상됩니다.

 

온라인 애니메이션 지원
WebP 파일 포맷은 오랫동안 밈(meme)을 비롯한 여러 온라인 애니메이션 파일 포맷의 표준이 된 GIF를 대신할 수 있습니다. 
Google에 따르면 애니메이션 WebP는 GIF보다 파일 크기가 훨씬 작고, 더 적은 바이트를 필요로 합니다.
WebP 파일의 장단점
WebP 파일 포맷은 장점과 단점이 명확하게 구분됩니다. 이를 잘 이해하고 활용하시기 바랍니다.

 

WebP 파일의 장점
웹 페이지가 빠르게 로드될수록 사용자 경험이 향상되고 검색 엔진에서 웹 사이트의 순위가 높아집니다. 
반면, 이미지 로딩 속도가 느린 웹 사이트는 사용자가 제품과 서비스를 구매할 때 이탈할 가능성이 큽니다.
WebP 포맷은 Google Chrome, Microsoft Edge, Mozilla Firefox 등 다양한 웹 브라우저와 호환됩니다.
WebP는 웹 사이트 이미지를 관리하기 쉬운 크기로 압축하므로, 파일 저장 공간이 여유롭습니다.
 

WebP 파일의 단점
WebP 파일 포맷은 인터넷을 염두에 두고 설계되었으므로 오프라인에서 이미지를 사용하는 경우 도움이 되지 않을 수 있습니다.
WebP 파일 포맷은 폭넓은 호환성을 가지고 있지만 Internet Explorer 등 일부 옛 브라우저의 경우, WebP 이미지를 지원하지 못할 수 있습니다.
약간의 압축으로도 이미지 품질이 저하될 수 있습니다. 고품질 이미지로 작업해야 하거나 전문 사진작가라면 이를 고려해야 합니다.
 

 

WebP 파일을 여는 방법
WebP 이미지는 기본 사진 애플리케이션보다 브라우저에서 보는 것이 더 빠릅니다.

 

WebP 파일을 열기 전에 먼저 호환되는 브라우저가 있는지 확인합니다. 설정이 완료되면 WebP 파일을 저장한 폴더를 열고 두 번 클릭합니다. 그러면 이미지가 기본 브라우저에서 자동으로 열립니다.

 

PNG, JPEG 등 기존 파일을 WebP 포맷으로 변환하는 옵션도 있습니다. 온라인에서 다양한 변환 툴을 찾을 수 있습니다.

 

 

WebP 파일을 생성 및 편집하는 방법
Adobe Photoshop을 사용하여 WebP 파일을 만들고 편집하려면 플러그인이 필요할 수 있습니다. 디바이스와 호환되는 신뢰할 수 있는 플러그인을 설치한 다음, 다음과 같이 파일을 생성할 수 있습니다.

 

Photoshop을 사용하여 이미지를 만들거나 편집한 다음, '다른 이름으로 저장'을 클릭합니다.

설정 패널이 표시되면 필요에 따라 파일을 추가로 편집합니다.
 

 

WebP 파일 관련 FAQ
WebP 파일과 JPEG 파일의 차이점은 무엇입니까?
두 파일 유형 모두 이미지를 공유하고 저장하기 쉽도록 압축합니다. 
그러나 WebP 파일은 일반적으로 기존 JPEG보다 훨씬 작습니다. 
Google 개발자에 따르면 손실 압축의 WebP 이미지는 같은 손실 압축 방식을 사용하는 
JPEG 파일보다 25%~34% 작을 수 있습니다.

 

WebP는 모든 브라우저에서 작동합니까?
수년간 개발과 진화를 거듭한 끝에 대부분의 웹 브라우저는 WebP 포맷을 지원합니다. 
그러나 Internet Explorer와 같은 일부 브라우저는 여전히 WebP 파일을 지원하는 데 어려움이 있을 수 있으므로
 Google의 최신 호환성 목록을 확인하는 것이 좋습니다.

 

WebP는 오픈 소스 파일 포맷입니까?
WebP는 오픈 소스 모델이므로 누구나 이 포맷을 작업하고, 성능을 개선할 수 있습니다.
 Google 개발자는 2010년 출시 이후 사용자의 의견을 적극 수렴하고 있습니다.

 

WebP 이미지의 최대 크기는 어떻게 됩니까?
최대 크기는 16,383 x 16,383픽셀입니다. WebP 이미지를 웹 사이트나 다른 위치에 추가하기 전에는 
최대 크기를 고려하는 것이 좋습니다.

 

WebP 이미지는 래스터 파일입니까, 벡터 파일입니까?
WebP 이미지는 래스터 파일의 일종입니다. 래스터는 벡터와 함께 이미지 파일의 주요 범주를 이룹니다.
 래스터 파일은 개별 픽셀로 구성되는 반면, 벡터 파일은 수식과 패스로 구성됩니다.
